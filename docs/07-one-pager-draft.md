## Draft copy
PayMyDine is a restaurant platform built to unify guest ordering and daily operations. Guests can order through mobile-friendly table/menu journeys, while teams manage menus, orders, reservations, kitchen workflows, staff permissions, and payments from one control layer.

With tenant-aware architecture and superadmin provisioning, PayMyDine is structured for multi-location growth. Payment orchestration includes multiple provider paths and webhook handling. Germany-focused compliance artifacts (including Fiskaly integration components) are present in code and can be validated per deployment.

**Core capabilities (proven in repository):**
- Tenant-aware platform architecture
- Menu, categories, modifiers, combos
- Table/QR ordering flows
- Reservations + kitchen display stations
- Staff roles, notifications, biometric/attendance modules
- Payment method/provider orchestration
