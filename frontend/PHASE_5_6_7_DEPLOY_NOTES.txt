PayMyDine frontend Phase 5/6/7 architecture cleanup delta.

Before copying these files on VPS, remove the old ownerless DOM repair folder:
  rm -rf /var/www/paymydine/frontend/features/customer-menu/legacy-dom-repairs

  npm run build
  node node_modules/typescript/bin/tsc --noEmit --pretty false
  npm run smoke:prod
  npm run checkout:safety
  npm run frontend:acceptance
